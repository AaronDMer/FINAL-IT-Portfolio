TO-DO MENU

This C++ program implements a simple text-based To-Do List application, allowing users to manage tasks through a menu-driven interface. It defines a Task structure containing an ID, description, and completion status. Tasks are stored in a std::vector. The program provides functionality to create new tasks, view the list of tasks, update task details (description and completion status), and delete tasks based on their ID.

The main function displays the menu and handles user input to call appropriate functions. Tasks are managed dynamically, ensuring IDs are unique and automatically assigned. The displayMenu function shows options, while helper functions (createTask, readTasks, updateTask, and deleteTask) perform respective operations with error checks like ensuring tasks exist before updating or deleting.
